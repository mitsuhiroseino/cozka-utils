## 関数命名ルール

| カテゴリ                     | 接頭辞            | 役割・期待される挙動                                         | 戻り値のイメージ    |
| :--------------------------- | :---------------- | :----------------------------------------------------------- | :------------------ |
| **存在保証**                 | **`ensure`**      | 入力が `null/undefined` でも代替値を返し、後続処理を壊さない | 非nullの値          |
| **安全実行**                 | **`maybe`**       | 入力が不適切なら処理をスキップしてそのまま返す               | 入力値              |
| **安全実行&保障**            | **`safe`**        | 入力が不適切なら処理をスキップしてデフォルト値を返す         | 処理後の値          |
| **再帰処理**                 | **`nested`**      | オブジェクトや配列の深い階層（子要素）まで走査して処理する   | 処理済みの構造体    |
| **判定**                     | **`is`**          | 状態を確認する                                               | `boolean`           |
| **判定(所有)**               | **`has`**         | 属性、所有の有無を確認する                                   | `boolean`           |
| **判定(提案)**               | **`should`**      | 「〜すべきか？」という条件判断（更新可否など）を行う         | `boolean`           |
| **変換(大)**                 | **`convert`**     | 型や単位の変換やデータ構造の組み替えなど、計算を伴う変換     | 変換後の値          |
| **変換(小)**                 | **`as`**          | 型の変わらない変化の小さな変換。意味は変わらない場合など     | 変換後の値          |
| **変換(文字列・ケース)**     | **`recaseTo`**    | 文字列のケースの変換                                         | 変換後の値          |
| **変換(文字列・形状)**       | **`retypeTo`**    | 文字列の形状(半角／全角、カナ／かな、数字／漢数字)の変換     | 変換後の値          |
| **変換(文字列・ノイズ除去)** | **`refine`**      | 文字列のノイズ除去・標準化（空白、改行、エスケープ）         | 変換後の値          |
| **変換(文字列・置き換え)**   | **`relabel`**     | 文字の置き換え。意味が変わる場合                             | 変換後の値          |
| **変換**                     | **`to`**          | 変換全般。分類に困ったものはここ。できるだけ避ける           | 変換後の値          |
| **値→表示用**                | **`format`**      | ユーザーに見せるための整形（日付、通貨、三桁区切りなど）     | `string`            |
| **表示用→値**                | **`unformat`**    | ユーザーに見せるための値からプログラムで扱う値に戻す         | 元の値              |
| **解析**                     | **`parse`**       | 文字列などの未構造データを、プログラムで扱える構造体に復元   | `Object` / `Date`等 |
| **生成**                     | **`create`**      | 新しいインスタンスやオブジェクトを生成して返す (Factory)     | 新規インスタンス    |
| **補間**                     | **`interpolate`** | 2点間の比率（ratio）に基づいた中間値を算出する               | 算出した数値等      |
| **制限(数値)**               | **`clamp`**       | 入力値が一定範囲を超えないように制限をかける                 | 範囲内の数値        |
| **制限(入力値)**             | **`limit`**       | 入力可能な値以外を受け入れないように制限をかける             | 制限された値        |
| **算出**                     | **`calc`**        | ロジックに基づき数値を算出する（純粋関数であることが多い）   | 計算結果            |
| **探索(追加)**               | **`find`**        | 条件に合うものを探す。見つからない可能性を含意する           | 目的の値 or `null`  |
| **取得(追加)**               | **`get`**         | 確実に存在するデータを取得する。なければエラーを投げる       | 目的の値            |
| **検証(追加)**               | **`assert`**      | 条件を満たさない場合に例外（Error）をスローする              | `void` (or Throw)   |
